# Папка для показа

Эта папка содержит упрощённый демонстрационный слой поверх основного репозитория.

Что здесь находится:
- `demo_main.ipynb` - главный ноутбук для показа. По умолчанию он читает уже сохранённые бэкапы, но при желании умеет создать новые прогоны и дообучить neural-модели.
- `demo_helpers.py` - вспомогательные функции для генерации сценариев, сохранения бэкапов, агрегации результатов и автоматических выводов.
- `configs/` - примеры YAML-конфигов для готовых preset-сценариев.
- `backups/` - сохранённые сигналы, checkpoints, таблицы результатов и картинки для быстрого показа без повторного обучения.

Логика работы:
1. Выбирается preset или вручную задаются параметры.
2. Ноутбук либо загружает готовый бэкап, либо создаёт новый прогон.
3. Для каждого прогона сохраняются:
   - `signals.npz`
   - `results.csv`
   - `summary.md`
   - `config_used.yaml`
   - `run_metadata.json`
   - checkpoints neural-моделей
4. В финальной части ноутбука строятся интерактивные графики и автоматически формируется текстовый вывод.

Главная идея показа:
- на простых каналах классические алгоритмы часто сильнее;
- на более тяжёлых и менее идеально смоделированных сценариях neural-подходы могут показать лучшую устойчивость.
